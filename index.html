"Alternative Pacman" World

Alternative Pacman is a 2-D grid where a pacman chases ghosts around a simple maze.

When a ghost is caught, it resets to a random positon on the board. Score = number of ghosts caught.

State

State = "px,py,rx,ry,yx,yy,bx,by,gx,gy"

  • px = x position of pacman
  • py = y position of pacman
  • rx = x position of redGhost
  • ry = y position of redGhost
  • yx = x position of yellowGhost
  • yy = y position of yellowGhost
  • bx = x position of blueGhost
  • by = y position of blueGhost
  • gx = x position of greenGhost
  • gy = y position of greenGhost

Action

Action = "i" where i takes the values:

  • ConrodWorld.ACTION_LEFT
  • ConrodWorld.ACTION_RIGHT
  • ConrodWorld.ACTION_UP
  • ConrodWorld.ACTION_DOWN

Score

Score = "s1" where

  • s1 = number of times a ghost was caught
